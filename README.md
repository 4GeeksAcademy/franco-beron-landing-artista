# Proyecto: Mi mundo en el dibujo

Este proyecto es una landing page para la artista Evelyn Barboza, diseñada para mostrar su universo artístico y su obra principal mediante una experiencia visual atractiva, moderna y accesible.

## Qué se implementó

1. Se creó `index.html` con la estructura completa de la landing page siguiendo el enunciado.
2. Se agregó `styles.css` para dar estilos modernos, ordenados y profesionales.
3. Se incluyó una sección HERO con la obra principal, un llamado a la acción para contacto por WhatsApp y navegación clara.
4. Se incorporó un menú de navegación con las siguientes secciones:
   - Galería
   - Sobre mí
   - Trayectoria
   - Servicio
   - Eventos
   - Preguntas frecuentes
   - Contacto
5. Se diseñó la sección de Galería con fotos de obras, nombre y descripciones de técnica, medidas y significado.
6. Se agregó información sobre técnicas usadas: pincel seco, stippling (punteado) y scumbling.
7. Se añadió el perfil de la artista, su inspiración, influencias artísticas, técnicas y herramientas.
8. Se documentó la trayectoria de la artista con hitos clave desde 2018 hasta 2025.
9. Se detallaron servicios ofrecidos: cuadros por encargo, clases para todas las edades y presentaciones en ferias y galerías.
10. Se incluyeron próximos eventos y proyecto de clases/presentaciones.
11. Se añadió una sección de Preguntas frecuentes con respuestas claras para visitantes y potenciales clientes.
12. Se colocó el contacto en el pie de página con Instagram y email.
13. Se usaron etiquetas HTML semánticas, atributos `aria`, `alt` en imágenes y datos estructurados de Schema.org para mejorar la accesibilidad y el SEO.

## Contenido principal de la landing

- Hero section con "Mi mundo en el dibujo", una obra destacada y llamado a la acción con contacto por WhatsApp.
- Navegación clara y ordenada.
- Galería con obras, títulos y descripciones de técnica, medidas e intención.
- Explicación de técnicas artísticas: pincel seco, stippling y scumbling.
- Sección "Sobre mí" con la biografía de Evelyn Barboza.
- Información sobre inspiración, influencias, técnicas y herramientas.
- Trayectoria artística con menciones de talleres y exposiciones.
- Servicios ofrecidos, incluidas clases y presentaciones.
- Próximos eventos/proyecto enfocado en clases y presentaciones.
- FAQ con preguntas clave sobre encargos, tiempos, técnicas, talleres y envíos.
- Footer con contacto por Instagram y email.

## Cómo ver el sitio

1. Instala las dependencias necesarias si aún no están instaladas:

```bash
pip3 install flask
```

2. Ejecuta el servidor local:

```bash
python3 server.py
```

3. Abre el navegador en `http://127.0.0.1:5000`
